Audio and video Formats
Type Format Description
Video MP4 A high-quality format with relatively small file sizes. MP4 is the most common video file type.
MOV Designed by Apple for the Quicktime player, MOV files are typically higher quality than MP4s.
WMV Developed by Microsoft for Windows operating systems, WMV files offer good quality with small file sizes.
Audio MP3 A compressed format that's popular for music because it has a manageable file size and only loses a small amount of sound data.
WAV An uncompressed format that preserves the original audio quality. It's the gold standard for studio recording.
AIFF An uncompressed format that's often used in professional audio. It's based on Pulse-Code Modulation (PCM).
